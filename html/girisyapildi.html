TÜM EĞİTİMLER

HTML5 ile Web Geliştirme

Bu eğitimde katılımcılar, HTML ve CSS kullanarak kişisel veya kurumsal düzeyde web siteleri geliştirebilecek becerileri kazanacaklardır. Eğitim, web tasarımının temellerine odaklanarak, HTML ile ilgili temel bilgiler ve kavramlarla başlayacaktır. Eğitim sonunda, katılımcılar kendi web sitelerini oluşturabilecek bilgi ve beceriye sahip olacaklardır.

Başlangıç Detayları Gör
CSS

Sürekli gelişen web teknolojilerinin en popüler ve en olmazsa olmaz dillerinden biriside CSS'dir. Eğitim dahilinde sıfırdan en ileri seviyeye kadar CSS'in yapısını, hangi alanlarda kullanıldığı, tüm özelliklerini ve tüm değerlerini en ayrıntılı şekilde ve bol örneklerle işlenmektedir.

İleri Seviye Detayları Gör
JavaScript

Sürekli gelişen web teknolojilerinin en popüler ve en olmazsa olmaz dillerinden birisi de Javascript'tir. Eğitim dahilinde sıfırdan en ileri seviyeye kadar Javascript'in mimarisini, hangi alanlarda kullanıldığı, tüm özelliklerini, tüm metotlarını, tüm fonksiyonlarını ve nesne tabanlı programlama yöntemlerini en ayrıntılı şekilde ve bol örneklerle işlenmektedir.

İleri Seviye Detayları Gör
Python Programlama

Python, modern yazılım geliştirme dünyasının en popüler dillerinden biridir. Bu eğitimde Python'ın temellerinden başlayarak, veri yapıları, fonksiyonlar, nesne yönelimli programlama ve web geliştirme gibi konuları öğreneceksiniz. Eğitim sonunda Python ile kendi projelerinizi geliştirebilecek seviyeye geleceksiniz.

Orta Seviye Detayları Gör
React ile Modern Web Geliştirme

React, modern web uygulamaları geliştirmek için en popüler JavaScript kütüphanelerinden biridir. Bu eğitimde React'ın temellerini, bileşen yapısını, state yönetimini ve modern web uygulamaları geliştirme tekniklerini öğreneceksiniz. Eğitim sonunda React ile profesyonel web uygulamaları geliştirebileceksiniz.
